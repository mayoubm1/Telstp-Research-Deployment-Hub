# TELsTP Security Guard Research Website
    
    ## Quick Start Guide
    
    1. **Extract this ZIP file**
    2. **Install dependencies:** `npm install`
    3. **Set up environment:** Copy `.env.local.template` to `.env.local` and add your Supabase credentials
    4. **Run locally:** `npm run dev`
    5. **Deploy to Vercel:** Push to GitHub and connect to Vercel
